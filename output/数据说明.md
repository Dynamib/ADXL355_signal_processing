## Step 0: 原始数据

### `01_raw_time_series.png`

**四子图：X / Y / Z 三轴时域波形 + 三轴叠加**

- 展示加速度计原始输出，采样率 1000Hz
- 每轴标注均值（重力分量）
- 可直观看到：X/Y 轴为水平静轴（均值≈0），Z 轴承载重力（均值≈-0.9g）
- 检查是否有明显的大幅冲击、数据丢失或异常跳变

### `09_fft_spectrum.png`

**三轴 FFT 幅度谱（去直流后）**

- 横轴：对数频率 (Hz)，纵轴：对数幅度 (dB)
- 直接对整段信号做 FFT，展示 |FFT(signal)| 的幅度谱
- **与 PSD 的区别**：FFT 幅度谱是"信号包含哪些频率，各多强"的直接回答；PSD 是功率的统计估计，侧重能量密度
- 观察：哪个频率处有**明显的峰值**（窄带成分），整体频谱的衰减趋势
- X/Y 轴幅度极低（接近本底噪声），Z 轴有明显频谱结构

## Step 1: 降噪与特征提取

### `02_psd_comparison.png`

**三轴 Welch 功率谱密度 (PSD) 对比**

- X 轴：对数频率，Y 轴：对数 PSD (g²/Hz)
- 每条曲线 = 一个轴的 raw（去直流）或 bandpass（带通滤波后）
- 观察：哪个频段能量最强、滤波是否有效衰减低频和高频噪声
- X/Y 轴 PSD 极低（接近噪声地板），Z 轴有明显频率成分

### `03_bandpass_filtered_X.png` / `Y` / `Z`

**每轴三子图：原始信号 + 带通滤波后信号 + PSD 对比**

- 子图1：原始信号（含重力直流分量）
- 子图2：带通滤波 1-200Hz 后的零均值信号
- 子图3：滤波前后 Welch PSD 叠加对比
- 验证滤波效果：低频呼吸干扰 (<1Hz) 和高频噪声 (>200Hz) 是否被有效抑制

### `04_envelope_kurtosis_X.png` / `Y` / `Z`

**每轴四子图：MED 盲反卷积 + 希尔伯特包络分析**

- 子图1：带通滤波后信号 + **峭度 (Kurtosis) 值**
  - 峭度衡量信号的"脉冲性"：高斯噪声≈0，冲击信号>>0
- 子图2：MED 盲反卷积输出 + 处理后的峭度值
  - **峭度变化越大 → 信号中蕴藏的脉冲特征越强**
  - Z 轴峭度通常从几十飙升到数千，表明存在大量被噪声掩埋的冲击事件
- 子图3：希尔伯特包络 — 信号瞬时幅度的上轮廓
- 子图4：对数稳定化包络 — 抑制重尾噪声中偶然大幅抖动的影响

> **核心指标**：MED 前后峭度差。差值越大，信号中脉冲成分（可能对应"得气"振动事件）越多。

## Step 2: 高分辨率时频分析

### `05_stft_X.png` / `Y` / `Z`

**STFT 短时傅里叶变换时频图**

- 横轴：时间 (s)，纵轴：频率 (Hz)，颜色：幅度 (dB)
- Hann 窗 512 点，75% 重叠，1024 点 FFT → 频率分辨率 ~2Hz
- 展示频率成分随时间的变化
- 观察：
  - 是否存在**持续窄带信号**（亮水平线）？
  - 是否存在**瞬态宽带冲击**（垂直亮条纹）？
  - Z 轴通常可见清晰的时变频率结构

### `06_sst_X.png` / `Y` / `Z`

**SST 同步压缩变换时频图**（降采样到 300Hz 后计算）

- 比 STFT 更清晰锐利的时频表示
- SST 将 STFT 中模糊的能量沿频率轴"压缩"到瞬时频率线上
- 横轴：时间 (s)，纵轴：频率 (Hz)，颜色：幅度 (dB)
- 观察：
  - 频率纹理是否在特定时刻出现/消失
  - 是否存在频率的**突然跳变**或**新频率成分出现**
  - 不同时间段的频率分布重心是否发生偏移

## Step 3: 统计验证与异常检测

### `07_sliding_window_features_X.png` / `Y` / `Z`

**四子图：2秒滑动窗口 (75%重叠) 频谱特征时间序列**

- 子图1：**频谱质心 + 带宽**
  - 质心 = 频谱的"重心频率"
  - 带宽 = 频谱围绕质心的散布宽度
  - 质心上升 → 高频能量占比增加；带宽扩大 → 信号变宽带
- 子图2：**频谱峭度**
  - >0 → 频谱呈尖峰状（窄带/纯音成分）
  - ≈0 → 频谱平坦（宽带噪声）
  - <0 → 频谱比高斯更平
- 子图3：**频带能量占比**
  - low (1-8Hz) / tremor (8-12Hz) / beta (12-30Hz) / gamma (30-100Hz) / high (100-200Hz)
  - 哪段频率主导信号？随时间是否变化？
- 子图4：**Z-score 异常检测**
  - 红色竖线标记 → 该窗口的频谱特征偏离整体中位数超过 3σ
  - X/Y 轴通常 0 异常（静轴），Z 轴可能有多处异常窗口

### `08_anomaly_heatmap_X.png` / `Y` / `Z`

**频谱特征 Z-score 热力图**

- 每行 = 一个频谱特征（质心、带宽、峭度、各频带能量比）
- 每列 = 一个时间窗口
- 颜色：红 = 偏离均值，蓝 = 低于均值
- 快速概览所有特征在时间轴上的整体偏离模式

### `anomaly_windows_X.csv` / `Y` / `Z`

**异常窗口数值表**

- 包含被 Z-score 标记为异常的所有窗口的完整频谱特征值
- 可用于回溯具体哪些时刻发生了"不寻常"的振动
- 列：t_center, centroid, bandwidth, spectral_kurtosis, E_low, E_tremor, E_beta, E_gamma, E_high

## 数据解读指南

| 你想知道 | 看的图 |
|----------|--------|
| 原始振动长什么样 | `01_raw_time_series.png` |
| 信号里有哪些频率（直接答案） | `09_fft_spectrum.png` — 看哪个频率有峰值 |
| 哪些频率在振动 | `02_psd_comparison.png` |
| 滤波去掉了什么 | `03_bandpass_filtered_*.png` |
| 有无脉冲特征（可能得气） | `04_envelope_kurtosis_*.png` — **看 Z 轴峭度变化** |
| 频率如何随时间变化 | `05_stft_*.png` → `06_sst_*.png`（更清晰） |
| 哪段时间不正常 | `07_sliding_window_features_*.png` 子图4 |
| 异常段的详细数值 | `anomaly_windows_*.csv` |

> **核心结论**：若 Z 轴 MED 后峭度大幅上升（数百到数千），说明原始振动信号中存在大量脉冲性冲击事件，这与艾灸"得气"振动（经络感传引起肌肉微动传递至艾条）的预期信号特征一致。
